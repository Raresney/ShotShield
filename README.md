# ShotShield

Find and redact sensitive data in screenshots — on your machine, before you share them.

> Status: early. Building the detection core first; the desktop and mobile apps come after it's solid.

## Why

We all paste screenshots into chats, issues and docs. Sooner or later one of them has
an API key, an IBAN, or a face in the corner. ShotShield scans the image locally,
highlights those regions, and lets you blur them in one step. Nothing leaves the device.
